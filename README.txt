Env_images contains images of generated environments.
Figures contains utility, policy and iteration plots for the experiments.
Results contains the csv version of figures (i.e., utility numbers and policy).
Report contains the report written for the assignment.

To run the entire script, run main.py. Note that both Parts 1 and 2 are run via main.py.

To view the environment generation process, refer to environment.py

To view the experiment design, refer to experiments.py.

To view the iteration algorithm, refer to the respective value_iteration.py or policy_iteration.py.
